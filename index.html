How it works

From download to graded

No login, no server, no installation. The worksheet is a single HTML file that runs entirely in Chrome.

01
Post to Google Classroom
Upload the HTML file as an attachment. Students download and open in Chrome — that's it.
02
Students work through activity
Draw Lewis structures, use PhET, fill in discovery tables, answer written questions. Auto-save keeps work in browser.
03
Students submit
Click "⬇ Submit Work" — downloads a snapshot HTML file. Turn in via Google Classroom or email.
04
Open the grader
Drag all student files onto the grader at once. Auto-scores geometry, angles, and MC. Export CSV for gradebook.
